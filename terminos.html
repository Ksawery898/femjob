Términos de uso y privacidad

Este texto es referencial y está incluido para completar el prototipo. No constituye asesoría legal.

1. Alcance del MVP

FemJob es un prototipo académico desarrollado para fines educativos. Las funcionalidades de postulación, guardado y contacto se encuentran en modalidad demostrativa y pueden no reflejar un servicio operativo real.

2. Uso responsable

Las usuarias deben utilizar la plataforma de manera responsable, evitando la carga de información falsa, contenido ofensivo o acciones que afecten la experiencia de otros usuarios.

3. Datos personales

En este prototipo no se almacenan datos. En una versión real, la plataforma requeriría:

  • Consentimiento informado para tratamiento de datos.
  • Medidas de seguridad y control de acceso.
  • Política de retención y eliminación de información.
  • Uso de datos agregados para métricas, evitando exposición de información sensible.

4. Recomendaciones y sesgos

El módulo de recomendaciones se presenta como concepto. En una implementación real, se deben considerar riesgos de sesgo algorítmico, transparencia y evaluaciones periódicas para evitar discriminación y asegurar pertinencia.

5. Contacto

Para consultas generales, empresas o instituciones, utiliza el formulario de contacto del prototipo.
